## 构建一个 mini_web_server ， 学习 Rust 用于 web server 的知识

最终完成的 server 绑定于 7878 端口。

可以使用 [http://127.0.0.1:7878/](http://127.0.0.1:7878/) 访问[^1]  

<br>

--- 

[^1]: 如果要让程序能上传到自己的 vps 上让外界访问，一般会要绑定到 0.0.0.0:8080 这样的 ip 和端口，根据具体的平台要求而不完全一样。
